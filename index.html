Research Question

Can GenAI reliably generate classroom-ready MCQs when guided by explicit pedagogical guardrails?

Classroom-ready = high quality + aligned to learning objectives

How We Evaluated Effectiveness

A question is classroom-ready when it passes both quality and relevance.

Generation Was Guided by a Carefully Designed Prompt

Quality guardrails
Well-built
  • One concept per question
  • Simple, plain language
  • Technically correct answers
  • Parallel answer choices
  • Avoid absolute terms
Relevance guardrails
On-target
  • Focus on important concepts
  • Emphasize core understanding
  • Avoid trivial edge cases

Prompt design was intentionally aligned with assessment best practices.

Absolute terms account for 97.5% of all accuracy failures.

Example: absolute terms
Why must the left side of an assignment be a variable in programming?
  1. Because only variables can store values
  2. Because expressions always return false
  3. Because variables can store multiple values simultaneously
  4. Because expressions are only used on the right side
Why it fails: “Only” overstates the case — constants, object fields, and array elements also hold values.

Example: correct answer is longest
How can a programmer prevent an infinite loop?
  1. Use a loop with no body
  2. Ensure the loop variable or condition changes to eventually become False
  3. Avoid using while loops
  4. Always start the loop condition as False
Why it fails: The correct answer is visibly longer than every distractor — a tell that students learn to follow.
