# MCQ Research - Visualization Files

All visualizations are now saved as **separate files** in PNG, PDF, and EPS formats at 600 DPI for publication quality.

## Quality Analysis Visualizations (10 files)

### Overall Comparison (4 files)
1. **quality_comparison_overall** - Bar chart comparing AI vs SME acceptance rates
2. **quality_distribution_ai** - Histogram showing failure count distribution for AI questions
3. **quality_distribution_sme** - Histogram showing failure count distribution for SME questions
4. **quality_comparison_table** - Summary table with counts and percentages

### By Discipline (4 files)
5. **quality_by_discipline_comparison** - Side-by-side bars comparing AI vs SME acceptance rates by discipline
6. **quality_by_discipline_ai** - Horizontal bars showing AI acceptance rates by discipline
7. **quality_by_discipline_sme** - Horizontal bars showing SME acceptance rates by discipline
8. **quality_avg_failures_by_discipline** - Average number of failures per question by discipline

### Failure Analysis (2 files)
9. **failure_by_criterion_ai** - Horizontal bars showing AI question failures by SAQUET criterion
10. **failure_by_criterion_sme** - Horizontal bars showing SME question failures by SAQUET criterion

## LO Alignment Visualizations (4 files)

1. **lo_alignment_by_discipline** - Stacked bars showing aligned vs not-aligned question counts
2. **lo_alignment_rate_by_discipline** - Percentage alignment rates by discipline
3. **lo_alignment_lowest_topics** - Top 10 topics with lowest alignment rates
4. **lo_alignment_summary_table** - Overall summary statistics table

## File Formats

Each visualization is available in three formats:
- **PNG** - High-resolution raster format (600 DPI) for presentations and web
- **PDF** - Vector format for high-quality printing and LaTeX documents
- **EPS** - Encapsulated PostScript for journal submissions and publications

## Usage Notes

- All charts use publication-quality formatting (600 DPI)
- Color palette is WCAG AA-compliant for accessibility
- Font sizes optimized for readability (10-12pt)
- Clean styling with minimal chart junk (no top/right spines)
- EPS files may show transparency warnings (this is normal and doesn't affect quality)
