<zyHTML id="ba872538-dad2-475d-dedf-0196476a2075" subsection="true" caption="Computer program basics">
<h3>Computer program basics</h3>
</zyHTML>


<zyHTML id="d8656552-2ea4-fbdd-811c-67fe1f45725a" caption="">
<p>Computer programs are abundant in many people's lives today, carrying out applications on smartphones, tablets, and laptops, powering businesses like Amazon and Netflix, helping cars drive and planes fly, and much more.  </p>

<p><zyDefn> A computer _program_ consists of instructions executing one at a time. </zyDefn> Basic instruction types are: 
<ul>
<li> <zyDefn> _Input_: A program gets data, perhaps from a file, keyboard, touchscreen, network, etc. </zyDefn> </li>
<li> <zyDefn> _Process_: A program performs computations on that data, such as adding two values like x + y. </zyDefn> </li>
<li> <zyDefn> _Output_: A program puts that data somewhere, such as to a file, screen, network, etc. </zyDefn> </li>
</ul>
 </p> 

<p> <zyDefn> Programs use _variables_ to refer to data, like x</zyDefn>, y, and z below. The name is due to a variable's value varying as a program assigns a variable like x with new values.  </p> 
</zyHTML>

Step 1: A basic computer program's instructions get input, process, and put output. This program first assigns x with what is typed on the keyboard input, in this case 2. Line, "x = Get next input", is highlighted and 2 appears in the input box. x is assigned the value 2.

Step 2: The program's next instruction gets the next input, in this case 5. Line, "y = Get next input" is highlighted and 5 appears in the input box. y is assigned the value 5.

Step 3: The program then does some processing, in this case assigning z with x + y (so 2 + 5 yields z of 7). Line, "z = x + y", is highlighted and z is assigned the value 7.

Step 4: Finally, the program puts z (7) to output, in this case to a screen. Line, "Put z to the output", is highlighted and 7 appears on the output screen.

<zyHTML id="11bc4636-c24b-60a3-28e9-56c5be34c6e1" caption="">
<p> <i> Mathematical thinking</i> became increasingly important throughout the industrial age to enable people to successfully live and work.  <zyDefn> In the information age, many people believe _computational thinking_,  or creating a sequence of instructions to solve a problem, will become increasingly important for work and everyday life.</zyDefn> <zyDefn>A sequence of instructions that solves a problem is called an _algorithm_.</zyDefn> </p>
</zyHTML>
