<zyHTML id="4f437938-60aa-4f43-8459-22bfe401c159">
<p> A simple Java program appears below. 
<ul>
<li> <zyDefn> A _program_ starts in main(), executing the statements within main's braces { }, one at a time.</zyDefn> </li>
<li><zyDefn> Each statement typically appears alone on a  line and ends with a _semicolon_, as English sentences end with a period. </zyDefn></li>
<li> The <zyICode>int wage</zyICode> statement creates an integer variable named wage. The <zyICode>wage = 20</zyICode> statement assigns wage with 20. </li>
<li> The print and println statements output various values. </li> 
</ul>

<zyAltDescription>Static figure:
Begin C++ code:
#include &lt;iostream&gt;
using namespace std;

int main() {
  int wage;

  wage = 20;

  cout &lt;&lt; &quot;Salary is &quot;;
  cout &lt;&lt; wage * 40 * 52;
  cout &lt;&lt; endl;

  return 0;

}
End C++ code.
When the integer variable wage is declared, wage is given a location in memory to hold a value. When wage is assigned with the value 20, the value 20 populates wage's location in memory.
The output is "Salary is 41600".

Step 1: A program begins executing statements in main(). 'int wage' declares an integer variable. 'wage = 20' assigns wage with 20. In memory, a memory address is assigned the variable name wage with the value 20.

Step 2: The cout statement outputs 'Salary is ' to the screen at the cursor's present location. Code line, "cout &lt;&lt; "Salary is ";", is highlighted and 'Salary is' is printed to the screen.

Step 3: This cout statement outputs the result of wage * 40 * 52, so 20 * 40 * 52 or 41600. Code line, "cout &lt;&lt; wage * 40 * 52;", is highlighted and 41600 is printed to the screen.

Step 4: This cout statement with 'endl' moves the output cursor to the next line on the screen.

Step 5: The 'return 0' statement ends the program.
