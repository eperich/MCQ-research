<zyHTML id="2747b353-324f-16ba-1159-3ef748a5247a" subsection="true">
<h3>Data exploration</h3>
</zyHTML>

<zyHTML id="87a55876-e4fa-e736-c29e-5e14bbd2dd12">
<p>
<zyDefn>_Data exploration_ is the first step of the data wrangling process, which includes exploring patterns and trends within a dataset.</zyDefn> Data exploration can be done visually through plots or figures, or numerically by comparing descriptive statistics.
</p>
<p>
<!--<zyDefn>_Data manipulation_ is the process of organizing or subsetting a dataset to explore a research problem.</zyDefn> Data manipulation-->During data exploration,  datasets may be split into multiple groups based on a categorical feature, or compared based on a specific condition. Descriptive statistics like the mean, median, or proportion can be calculated and compared across groups or conditions. 
</p>
</zyHTML>

<zyQChoice>
<zyQAns>Data <!--manipulation-->exploration results in a larger dataset.</zyQAns>
<zyQExpl>Data exploration includes finding patterns and trends within a dataset, without adding new features or instances to the original data.</zyQExpl>
</zyQChoice>
<zyQChoice>
<zyQAns>Data <!--manipulation-->exploration creates new features in a dataset.</zyQAns>

<zyQ>
<zyQText>Which research question requires <!--data manipulation-->splitting a dataset into groups to answer?</zyQText>
<zyQChoice>
<zyQAns>Is there a relationship between a car's gas mileage and engine size?</zyQAns>
<zyQExpl>Gas mileage and engine size are both numerical features. The relationship between numerical features can be explored in a line or scatter plot, so <!--data manipulation-->splitting a dataset is not needed.</zyQExpl>
</zyQChoice>
<zyQChoice correct="true">
<zyQAns>Is there a relationship between a car's gas mileage and drive type?</zyQAns>
<zyQExpl>For each drive type, gas mileage may be described using a statistic like the mean or median.</zyQExpl>
</zyQChoice>
<zyQChoice>
<zyQAns>Is there a relationship between a car's gas mileage and number of cylinders?</zyQAns>
<zyQExpl>Gas mileage and number of cylinders are both numerical features. The relationship between numerical features can be explored using a line plot or scatter plot, so <!--data manipulation-->splitting a dataset is not needed.</zyQExpl>
</zyQChoice>
</zyQ>

<zyQChoice>
<zyQAns>Manufacturer</zyQAns>

</zyQChoice>
<zyQChoice correct="true">
<zyQAns>Drive</zyQAns>
<zyQExpl>Grouping the cars dataset based on drive would result in three subsets: all-wheel drive, front-wheel drive, and rear-wheel drive.</zyQExpl>

<zyQAns>Liters</zyQAns>

<zyHTML id="512f1a3d-936c-1c57-baa0-98b1ca4fabb6" subsection="true">
<h3>Grouping data</h3>
</zyHTML>

<zyHTML id="a282e2e5-dda8-21c3-8888-2975a4e3717d">
<p>
<zyDefn>_Grouping_ is used to separate a dataframe into subsets based on levels of a categorical feature.</zyDefn> In some cases, a different analysis or model may be applied to each group. In other cases, grouping may be a temporary operation for calculating group sizes or descriptive statistics like group means. <zyDefn>A _frequency table_ is a table containing group sizes for a categorical feature.</zyDefn>
</p>
</zyHTML>

<h3>Pivot tables</h3>
</zyHTML>

<zyHTML id="ef28b338-3a5e-3baa-a672-21da9efa023c">
<p>
<zyDefn>A _pivot table_ calculates and displays descriptive statistics after grouping based on values of two categorical features.</zyDefn> One categorical feature is assigned to the pivot table's rows, and another categorical feature is assigned to the columns. <zyDefn>A _contingency table_ is a special case of a pivot table in which the descriptive statistic is the number of instances in each combination of categorical features.</zyDefn>
</p>

<zyHTML id="73b9c6de-6ccd-b267-512a-4f85e605c341">

<p>
The <zyICode>pandas</zyICode> package contains two useful methods for data <!--manipulation-->exploration. <zyICode>df.groupby()</zyICode> splits a dataframe <zyICode>df</zyICode>  into subsets, and <zyICode>df.pivot_table()</zyICode> creates pivot tables based on two categorical features. Both <zyICode>df.groupby()</zyICode> and <zyICode>df.pivot_table()</zyICode> can be combined with <zyICode>pandas</zyICode> descriptive statistics methods.
</p>

<p>
<zyICode>df.groupby()</zyICode> sets the grouping feature using the <zyICode>by</zyICode> parameter. Missing values for the grouping feature can be removed by setting <zyICode>dropna=True</zyICode> or grouped into a separate category by setting <zyICode>dropna=False</zyICode>. Additional parameters for <zyICode>df.groupby()</zyICode> can be found in the <zyLink link="https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.groupby.html">group by documentation</zyLink>.
</p>

<p>
<zyICode>df.pivot_table()</zyICode> takes several parameters. The <zyICode>value</zyICode> specifies the values in the pivot table's elements. The feature in the pivot table's rows  is specified using <zyICode>index</zyICode> and the feature in the pivot table's columns is <zyICode>columns</zyICode>. <zyICode>aggfunc</zyICode> specifies a function to apply to the values in each row/column combination within the pivot table. The default aggregate function is <zyICode>np.mean</zyICode>.
</p>
